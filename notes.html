Lesson 1

What is HTML?

HTML stands for HyperText Markup Language,
It is made up of:

1. Text Content, what you see
2. Markup, what it looks like
3. Reference to, e.g. images and windows
4. Other Documents, links to other pages

Tags, Elements, Attributes
Here are some terms to keep in mind
1. Tag: An HTML tag contains content with angled
brackets with close tags or void tags that don't
require a close tag.
2. Element: An HTML element refers to everything
within a set of opening and closing tags
3. Attributes: This is a property of an HTML element.
for example, to set the href attribute of an anchor tag
to the udacity URL, you would write <a href="www.udacity.com">
Inline vs Block
-<b> = inline
- Ex: text <br> text
- More inline: <b>, <em>, <img>
-<p> = block
- Ex: <p> text </p>
- Text can also be contained in a box with dimension and width

<span> text </span> is also an inline tag, which can be assigned attributes such as "I have <span style:="color:blue;"> blue </span> eyes.
<div> text </div> is a block tag, which can also be assined attributes using CSS, like <div class="bar">

For now, lets talk about why CSS is "Cascading."

-Think about it like U.S. laws:

National law < State law < Municipality law < House rules in CSS it is opposite, "House rules" apply to "National law" in the way that the most specific rule applies, meaning the rule that best fits the element applies
CSS provides a way to avoid repetition when coding by using selectors. By avoiding repetition you save time, your code and style is more consistant, there is less code to write, and you are less likely to make errors.
Selectors

As we get into selectors here are some terms to help you understand:

1. Inheritance : properties are applied not only to the specific element, but also to its descendants. Box-related properties are not inherited, inherited properties include color, font, letter spacing, line height, list-style, text-align, text indent, text transform, visibility, etc.
2. Selector : defines which elements of the page that the styling will apply to.
3. Style : can refer to a few things such as the HTML element, HTML attribute, the general look of the webpage, or as a verb (I need to style my webpage).
4. Rule : line of CSS code describing a value that a certain attribute should take on.
5. Property : the property you want to change.
5. Value : the value you want to assign to another attribute.
6. Selctor(2) : the name you sue to target elements assigned to a class or i.d. attribute in HTML.
7. Class : a group of elements styled together.
8. I.D. : a unique type of selector.

A "Class" selector will style all the content with a div matching the class name.

Example:

.dessert {
font-size: 50px;
}

<div class="dessert">
    froyo is hella good
</div>

How Do I Include CSS Style?

The best method to include your CSS style is to attach a separate CSS file to your HTML doc in the head of your HTML file.

Example:

<head>
    <link rel="stylesheet" href="main.css">
</head>

Depending on the parent-child element relationship you want, you will use different selectors. An inherited selector will give the value to the direct child of the parent element and the in-direct child elements of the parent element. A non-inherited selector will only give the value to the direct child element of the parent element.
The Box Model

*{
box-sizing:
border-box:
}

Includes border and padding but not the margin! And since box sizing in CSS is relatively new, browser specific prefixes must be applied:

*{
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;
box-sizing: boarder-box;
}

Percentage values change based on screensize while pixels do not.
Also, setting a max-width will allow the box to get smaller than necessary but not larger than the max width.
